# Controler & Driver


## [Controler Board](https://www.aliexpress.com/item/1005004625846519.html?spm=a2g0o.order_list.order_list_main.11.7af81802aPO39Z#nav-specification)

* Software download:
  * [Macrobase-tech/CNC-Software](https://github.com/Macrobase-tech/CNC-Software)
* Packing List:
  * 6 axis Board-Z Tool included
  * 1 x CNC Board
  * 1 x USB
  * 1 x Z Probe
* Button Function:
  * s1 is reset,s2 to download firmware.
  * s1 connects to EN,s2 connects to Boot.
